# [PERSONAL PROJECT] - GAME OẲN TÙ TÌ (ROCK PAPER SCISSORS)

## Giới thiệu
Dự án này là một sản phẩm cá nhân (Personal Project) được thực hiện với mục đích **luyện tập và củng cố kỹ năng lập trình JavaScript**.  
Mục tiêu chính là rèn luyện tư duy logic, cách xử lý sự kiện (Event Handling) và thao tác với DOM (Document Object Model) thông qua việc xây dựng trò chơi Oẳn Tù Tì kinh điển.  
Hệ thống mô phỏng trận đấu giữa Người chơi và Máy (Bot) với kết quả ngẫu nhiên.

## Công nghệ sử dụng
* **Ngôn ngữ:** * HTML5 (Cấu trúc trang)
    * CSS3 (Flexbox, Animation, Responsive cơ bản)
    * JavaScript (ES6 - Xử lý logic game)
* **Công cụ:** VS Code
* **Triển khai:** Chạy trực tiếp file HTML hoặc Live Server

## Tính năng chính
1. **Chơi với máy:** Người chơi chọn Kéo, Búa hoặc Bao; máy sẽ chọn ngẫu nhiên.
2. **Xử lý logic:** Tự động so sánh kết quả để đưa ra thông báo Thắng, Thua hoặc Hòa.
3. **Giao diện tương tác:** Hiệu ứng hover, hiển thị kết quả trực quan bằng hình ảnh và màu sắc.
4. **Chức năng Replay:** Nút "Chơi lại" giúp reset game nhanh chóng mà không cần F5 thủ công.

## Hướng dẫn cài đặt 
1. **Clone repository**:
```bash
https://github.com/KOALAFixBug/Game-Rock-Paper-Scissors.git
```
2. **Di chuyển vào thư mục**:
```bash
cd Rock_Paper_Scissors_Game
```
3. **Chạy dự án**

**Cách 1:** Mở trực tiếp file HTML

Chỉ cần tìm đến file **index.html** (hoặc file HTML chính của bạn) trong thư mục dự án.

Nháy đúp chuột vào file đó, trang web sẽ tự động mở lên bằng trình duyệt web mặc định của bạn.

**Cách 2:** Sử dụng **"Live Server"** (Khuyên dùng cho phát triển) Nếu bạn sử dụng trình soạn thảo code như Visual Studio Code, cách tốt nhất là dùng extension "Live Server":

Cài đặt extension Live Server từ VS Code Marketplace.

Sau khi cài đặt, bạn chỉ cần nhấn chuột phải vào file index.html và chọn **"Open with Live Server"**.

Trang web sẽ tự động mở trong trình duyệt và tự động tải lại mỗi khi bạn lưu file, giúp việc code nhanh hơn rất nhiều.
